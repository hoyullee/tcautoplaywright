|------|-----|

## 용어 설명
- **채용 홈**: 원티드 메인 채용 페이지 (https://www.wanted.co.kr/)
- **탐색 페이지**: GNB의 채용 탭을 클릭하면 진입하는 페이지 (https://www.wanted.co.kr/wdlist)
- **이력서 페이지**: GNB의 이력서 탭을 클릭하면 진입하는 페이지
  - 비로그인: https://www.wanted.co.kr/cv/intro
  - 로그인: https://www.wanted.co.kr/cv/list
- **교육이벤트**: GNB의 교육•이벤트 탭을 클릭하면 진입하는 페이지 (https://event.wanted.co.kr/)
- **콘텐츠**: GNB의 콘텐츠 탭을 클릭하면 진입하는 페이지 (https://www.wanted.co.kr/events)
- **소셜**: GNB의 소셜 탭을 클릭하면 진입하는 페이지 (https://social.wanted.co.kr/community)
- **프리랜서**: GNB의 프리랜서 탭을 클릭하면 새 페이지에서 진입되는 페이지 (https://www.wanted.co.kr/gigs/experts)
- **포지션 상세 페이지**: 탐색 페이지에서 포지션 카드를 선택하면 진입되는 페이지
  - URL 패턴: `https://www.wanted.co.kr/wd/{포지션ID}` ({포지션ID}는 포지션 카드별로 고유하게 부여된 숫자)
- **GNB**: Global Navigation Bar, 페이지 상단 글로벌 네비게이션 바
- **LNB**: Local Navigation Bar, 페이지 좌측 또는 서브 네비게이션 바
