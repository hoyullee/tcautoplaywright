## 테스트 케이스 정보
- 번호: 61
- 환경: PC
- 기능영역: GNB > 이력서
- 사전조건: 1. 로그인 상태
2. 이력서 탭 진입 상태
- 확인사항: 1. 내 이력서 리스트 영역
2. '작성 중' 상태의 이력서 카드
3. 이력서 카드 우측 하단
4. 3점(더보기) 버튼 선택
5. 이력서 삭제 버튼 선택
6. 삭제 확인 팝업 > '삭제' 선택
- 기대결과: 아래 툴팁 노출되며 이력서 카드 삭제됨
 - '삭제되었습니다.'

- 세션: 사전조건이 '로그인 상태'이므로 직접 로그인하지 말고 저장된 세션 파일을 로드하세요: storage_state='work/auth_state.json'

## 작업
1. `test/test_61_working.py` 생성 후 코드 작성
2. `python3 test/test_61_working.py` 실행
3. 성공 시 `test/test_61_success.py`로 이름 변경, 실패 시 수정 후 재시도
4. 마지막에 반드시 `AUTOMATION_SUCCESS` 또는 `AUTOMATION_FAILED: 에러메시지` 출력
